Voice Converter

The Voice-to-Text Converter is a React.js application designed to leverage the capabilities of the Web Speech API to transcribe spoken words into text in real-time. It includes intuitive features for managing transcriptions, such as saving, editing, downloading, and deleting them. The app provides a simple, responsive interface, making it suitable for various devices.


  1. 1. Voice-to-Text Conversion
    - Converts spoken words into text using the Web Speech API.
    - Supports real-time transcription with continuous updates as you speak.

  2. 2. Save Transcriptions
    - Save transcriptions with a timestamp for easy organization.

  3. 3. Manage Saved Texts
    - Dropdown Options for Each Saved Text:
    = Download: Save the text locally as a .txt file.
    = Edit: Modify or add new content to the saved text.
    = Delete: Remove the transcription from the saved list.
    = Each saved text is accompanied by the date and time it was created.

  4. 4. Clear Current Text
    - Quickly clear the text area for a fresh start.
